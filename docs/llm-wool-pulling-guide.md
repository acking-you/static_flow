## 一、为什么不用国产 Coding Plan？

不是不想支持国产，是真的没法用。

2026 年 Kimi、智谱纷纷推出 Coding Plan，价格看似便宜，但一算账就会发现：**便宜的额度不够用，够用的价格不比 OpenAI 便宜，而能力还差一大截。**

| 产品             | 套餐与月费                                                  | 实际体验                                                                                                                            |
| ---------------- | ----------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Kimi K2.5 Coding | Andante ¥49 / Moderato ¥99 / Allegretto ¥199 / Allegro ¥699 | SWE-bench 76.8% 跑分好看，但实际工程场景 agent 规划能力和 Codex 差距明显；想正经用至少 ¥99–199/月，高档 ¥699 直逼 OpenAI Pro        |
| 智谱 GLM Coding  | 基础版 ¥20/月起，季度/年度有折扣                            | 基于 GLM-5 / GLM-4.7，号称比 Copilot 便宜 85%，但复杂工程任务（多文件重构、长链路 debug）远不如 GPT-5.4；近期频繁涨价，社区怨声载道 |

核心问题：
- **跑分好看 ≠ 实际好用**：多文件重构、复杂 debug、架构设计这些真实工程场景，国产模型和 GPT-5.4/Claude Opus 4.6 差距巨大
- **Agent 能力差距大**：Codex 已经能自主执行多步骤工程任务，国产 agent 在长链路任务中经常跑偏或中断
- **涨价没商量**：智谱 GLM Coding Plan 近期多次涨价，老用户被迫补差价续费

但更关键的是——大善人 OpenAI 层出不穷的薅羊毛机会，让你总能以最舒服的方式用上最顶级的模型。就算某条路断了，还有其他路径可以无缝切换，这种"薅不尽"的体验是国产方案完全没有的。

## 三、薅羊毛四大路径

### 路径 1：Business 免费名额（低门槛，有封号风险）

OpenAI 活动期间，每个账号可免费领取 5 个 Business（Teams）席位，具备完整 GPT-5.4 使用权限。

操作：注册账号 → Teams 管理页领取免费席位 → 自用或挂闲鱼（约 ¥5/个）。

近期 OpenAI 大规模封号，存活率下降，适合短期薅一波，不适合长期依赖。风险 ⚠️ 中高。

---

### 路径 2：免费号池 + 中转站（零成本，技术门槛稍高）

OpenAI 免费账号注册限制不严，可以通过自动化工具批量注册，汇聚额度组成 API 中转池。

代表项目：[CPA-Codex-Manager](https://github.com/Maoleio/CPA-Codex-Manager) —— 自建免费号池，自动管理账号生命周期、额度轮转、失效替换，零成本。

操作：Clone → 配置 → 启动后自动注册管理 → 对外暴露 OpenAI 兼容 API。

优势是完全零成本、自建自用；劣势是需要一定技术基础（Docker / Node.js），且免费账号模型权限有限。风险 ⚠️ 中。

---

### 路径 3：活动免费开 Plus/Pro（性价比最高）

部分账号满足特定条件后，可触发免费开通一个月 Plus 甚至 Pro。条件因时而异，通常与注册时间、地区、使用历史有关，需关注社区最新情报。

闲鱼行情：Plus 约 ¥20–30，Pro 约 ¥30–50。获得完整权限，包括 GPT-5.4、DALL·E、Codex 全部能力。

这是目前性价比最高的路径。如果薅到 Pro，还可以 5–6 人共用，分摊成本无限畅享。风险 ⚠️ 低中。

---

### 路径 4：汇率差购买（成本最高，但最稳）

不同地区的 OpenAI 订阅价格存在汇率差异，从最便宜的地区购买可显著降低成本。

操作：使用目标地区支付方式（虚拟信用卡 / 当地支付渠道）→ 切换地区购买订阅。

成本在所有路径中最高，但最稳定、不依赖任何活动或漏洞，适合长期使用。风险 ✅ 低。

## 四、拿到账号后怎么用？

拿到 Plus/Teams 账号后，最直接的用法是登录 Codex CLI 使用。但有几个痛点：

| 痛点           | 说明                                                    |
| -------------- | ------------------------------------------------------- |
| 每次都要开代理 | OpenAI 在国内无法直连，每次使用前必须挂梯子             |
| 额度不够用     | Plus/Teams 额度对重度用户远远不够，需要多个账号来回切换 |
| 封号风险       | 账号随时可能被封，又得重新搞号                          |

当然直接登录也有优势：原生 WebSocket 协议延迟更低、包体更小，以及 Codex 的 [Remote Compact](https://ackingliu.top/posts/codex-compact-local-and-remote-deep-dive) 功能只有原生登录才能用。
